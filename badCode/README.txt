Данный проект представляет собой библиотеку для анализа текста на C#.
Проект полностью функционален и выполняет все заявленные задачи, но содержит множество архитектурных недостатков.
Основные возможности: подсчет количества слов в тексте несколькими алгоритмами, поиск самого длинного слова различными методами, анализ частоты использования символов
сохранение истории операций с ограничением размера, различные форматы вывода результатов анализа
